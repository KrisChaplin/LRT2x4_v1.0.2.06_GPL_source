Edit User Password

To edit User settings, click Edit to the right of the user name in the Access >> Users & Groups window. The Edit User Settings window will be displayed.

 

The general user information, including the User Name, Description, and Domain Name will be displayed. These fields are not configurable. If information supplied in these fields need to be modified, then remove the user by clicking Delete User and then recreate the user with the correct information.

 

The user password may be modified. To change the user password:

 

1. Enter the new user password in the Password field.
 

2. Click Submit to update the configuration.
