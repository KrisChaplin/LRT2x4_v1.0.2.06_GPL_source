Terminal Services Applications

The Applications window enables you to access Terminal Services applications hosted on Windows desktops and servers. You may also access this window by clicking the Application option in the left navigation menu.

 

Note: Some menu options, including Applications, Services and My Desktop, may be disabled by the SSL VPN administrator.  

 

Terminal Services

 

Terminal Services or RDP (Remote Desktop Protocol) is a protocol supported on Windows XP Professional and Windows 2000 and 2003 Server. Once Terminal Services has been configured, users can connect to the machine remotely, access files and software, save file changes and perform other advanced features. Terminal Services also allows multiple users to access a single machine simultaneously.

 

Terminal Services supports the unique ability to launch individual applications running on a remote desktop or server. Just running a single application reduces the latency and overhead compared to accessing the entire remote desktop.

 

Terminal Services Applications

 

The EasyAccess SSL VPN portal includes pre-defined or custom Terminal Services applications that users may launch from the SSL VPN portal.

 

EasyAccess includes an ActiveX Terminal Services client. The ActiveX client combines the redistributed Microsoft® TSWeb control with an EasyAccess proprietary SSL encryption and authentication control. The ActiveX client is compatible with Microsoft RDP version 5.0 and includes advanced features such as sound and printer services.
