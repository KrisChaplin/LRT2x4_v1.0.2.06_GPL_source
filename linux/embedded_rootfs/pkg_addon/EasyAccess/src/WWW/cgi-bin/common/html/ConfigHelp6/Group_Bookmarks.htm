Group Bookmarks

To add or edit Group Bookmarks, click the group name in the Access >> Users and Groups window. The Edit Group Settings window will be displayed.

Edit Group Bookmarks

EasyAccess bookmarks provide a convenient way for SSL VPN users to access computers on the local area network that they will connect to frequently. Group bookmarks will apply to all members of the specific group. To define group bookmarks, click Add Bookmark. An Add Bookmark window will be displayed.
 

When group bookmarks are defined, all group members will see the defined bookmarks from the EasyAccess SSL VPN portal. Individual users will not be able to delete or modify group bookmarks.
 

  1. To create a bookmark, enter a descriptive name in the Bookmark Name field.
     

  2. Enter the domain name or the IP address of a host machine on the LAN in the Name or IP Address field.
     

  3. Select the service type in the Service menu.
     

  4. Click Submit to update the configuration.

 

Once the configuration has been updated, the new group bookmark will be displayed in the Edit Group Settings window.
