Virtual Passage Requirements for MacOS X

To view the Virtual Passage window for Mac operating systems, click the Virtual Passage option in the left navigation menu. Then click the MacOS X option.

 

Virtual Passage Overview

 

Virtual Passage enables remote users to securely connect to the remote network. With Virtual Passage, SSL VPN users virtually join the remote network: users can mount network drives, upload and download files, and access resources in the same manner as if they were on the local network.

 

Virtual Passage Requirements for MacOS Users

 

  • MacOS X operating system

  • Safari 1.0 and greater

  • To initially install the Virtual Passage software, the user must know the administrator password for the system.

  • Java must be enabled on the browser
