Users

To view the EasyAccess Users & Groups window, click the Access option in the left navigation menu, then click the Users & Groups option.

Add a New User

To create a new group, click Add User. An Add User window will be displayed.
 

  1. Enter the user name for the user in the User Name field. This will be the name the user will enter in order to log into the EasyAccess SSL VPN portal.
     

  2. Select the name of the group to which the user belongs in the Group menu.
     

  3. Click Submit.

 

If the selected group is in a domain that uses external authentication, such as Active Directory, RADIUS, NT Domain or LDAP, then the Add User window will close and the new user will be added to the Users and Groups table.

NOTE: Groups configured to use Radius, LDAP, NT Domain or Active Directory authentication do not require passwords because the external authentication server will validate user names and passwords.
 

It is only necessary to enter RADIUS, LDAP, NT and Active Directory user names if you wish to define specific policies or bookmarks per user. If users are not defined in EasyAccess, then global policies and bookmarks will apply to users authenticating to an external authentication server.
 

If the selected group is in a domain that uses internal database authentication, such as the default "EasyAccess" domain, then define the user password and user type.

 

  1. Enter the user’s password in the Password field.
     

  2. Re-enter the password in the Confirm Password field.

    NOTE:     Both the user name and password are case-sensitive.
     

  3. Select the user type, either User or Administrator from the User Type menu.
     

  4. Click Submit to update the configuration.

 

Once the user has been added, the new user will be added to the Users and Groups window.

Delete a User

To delete a user, click the user name that you wish to remove. The Edit User Settings window will be displayed. In the Edit User Settings window, click Delete User. Once deleted, the user will be removed from the Users and Groups window.

 

Edit a User

 

The Edit User Settings table shows the User Name, Group Name, and Domain Name. These fields are not configurable. If information supplied in these fields need to be modified, then remove the user by clicking Delete User and then recreate the user with the correct information.

 

If the user authenticates to an external authentication server, then the User Type and Password fields will not be shown. The password fields are not configurable because the authentication server will validate the password. The user type is not configurable because users that authenticate to an external AAA server cannot have administrative privileges.

 

The user password may be modified. To change the user password:

 

  1. Enter the new user password in the Password field.
     

  2. Click Submit to save the configuration changes.
     

To change the user inactivity timeout:
 

  1. Enter the number of minutes of inactivity to allow in the Inactivity Timeout field.
     

  2. Click Submit to save the configuration changes.
     

To change the user connection limit:

 

     1.  Enter the number of connections to be allowed in the Connection Limit field.

 

     2.  Click Submit to save the configuration changes.
