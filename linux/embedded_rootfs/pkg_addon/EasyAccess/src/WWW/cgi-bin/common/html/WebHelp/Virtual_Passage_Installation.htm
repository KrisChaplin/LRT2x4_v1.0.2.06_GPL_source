Virtual Passage Installation for Windows

To install Virtual Passage and connect to the SSL VPN gateway, perform the following instructions:

 

  1. Click the Create a Virtual Passage icon to open a connection to the SSL VPN gateway. The EasyAccess Virtual Passage will transparently connect to the SSL VPN gateway and remote network. The connection process will take between 5 to 15 seconds.

 

NOTE: Virtual Passage requires Internet Explorer. ActiveX content must also be supported. If you are running Windows XP SP2, you may see a warning message when you attempt to open a Virtual Passage connection. Right click on the warning message and select Accept to allow the Virtual Passage software to install.

 

When the connection is established, a Virtual Passage icon in the Windows Taskbar Notification Area will appear.

 

To end the SSL VPN Virtual Passage connection, open the Virtual Passage Properties by selecting the Virtual Passage icon in the Windows Taskbar Notification Area. Then click Disconnect. The Virtual Passage connection will take approximately 10 seconds to terminate.

 

NOTE: SSL VPN users can only create a Virtual Passage to the remote network during an active SSL VPN session. Once a user logs out of the SSL VPN portal or closes the portal window, the Virtual Passage session will terminate. In order to create a new Virtual Passage connection, the user must log back into the SSL VPN portal and relaunch the Create a Virtual Passage icon.
