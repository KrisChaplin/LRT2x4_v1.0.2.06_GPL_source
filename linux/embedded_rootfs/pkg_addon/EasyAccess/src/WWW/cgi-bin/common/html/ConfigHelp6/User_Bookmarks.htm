User Bookmarks

To edit User settings, click Edit to the right of the user name in the Access >> Users and Groups window. The Edit User Settings window will be displayed.  

Edit User Bookmarks

To define user bookmarks, click Add Bookmark. An Add Bookmark window will be displayed.
 

When user bookmarks are defined, the user will see the defined bookmarks from the EasyAccess SSL VPN portal. Individual user members will not be able to delete or modify bookmarks created by the administrator.
 

  1. To create a bookmark, enter a descriptive name in the Bookmark Name field.
     

  2. Enter the domain name or the IP address of a host machine on the LAN in the Name or IP Address field.
     

  3. Select the service type in the Service menu.
     

  4. Click Submit to update the configuration.

 

Once the configuration has been updated, the new user bookmark will be displayed in the Edit User Settings window.
