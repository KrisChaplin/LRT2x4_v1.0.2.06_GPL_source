Portal Layouts

To view the Portal Layouts window, click Access and then Portal Layouts in the left navigation menu.

Configuring EasyAccess Portal Layouts

The EasyAccess administrator may define individual layouts for the EasyAccess portal. The layout configuration includes the theme, menu layout, portal pages to display, portal application icons to display, and web cache control options.

  

A default portal layout is the EasyAccess portal. Additional portal layouts can be added and modified.

 

NOTE: To apply a portal layout to a domain, add a new domain and select the portal layout from the Portal Layout Name menu on the domain configuration page. The selected portal layout will be applied to all users in the new domain.
