Firewall

To view the EasyAccess Firewall window, click the EPS option in the left navigation menu, then click the Firewall option.

 

Add a Firewall

 

To create a new Firewall, click Add Firewall. An Add Firewall window will be displayed.

 

          1. Enter the Firewall name and version in the Firewall name and Version fields that should be present in the

              EPS Client to logging in to EasyAccess.

          2. Select the enable version check and check for firewall enabled.

          3. Click Submit.

 

          Note: The following Antivirus and its versions were supported in EasyAccess.

 

                   McAfee Personal Firewall  6.0.6014, 6.1.6144, 8.2

                   TrendMicro 14, 15.30.1151

                   Kaspersky 6.0.2.621, 6.0.2.678

 

Add a Firewall List

 

To create a new Firewall List, click Add Firewall List. An Add Firewall List window will be displayed.

 

          1. Enter the FirewallList in the Name field which contains list of Firewall to be defined for EPS policy.

          2. Select the various Firewall Names from the multiple select combo box .

          3. Click Submit.
