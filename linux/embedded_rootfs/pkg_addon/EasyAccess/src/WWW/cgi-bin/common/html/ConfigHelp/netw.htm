Interfaces

The IP settings and interface settings of the EasyAccess appliance may be configured through the Network >> Interface window. From the Interface window, an EasyAccess administrator can select the interface to terminate SSL VPN connections and then configure IP settings per network interface.

 

Example Configuration for EasyAccess SSL VPN

 

An example of a network configuration where the EasyAccess appliance is deployed as a standalone SSL VPN device. A separate access router or firewall performs perimeter security.

 

  • LAN (internal) IP address: 192.168.0.2

  • LAN subnet: 192.168.0.0/24  (subnet mask: 255.255.255.0)

  • LAN gateway address (Firewall/Router address): 192.168.0.1

 

In this configuration, the IP addresses of devices in the local network should be configured in the 192.168.0.0/24 subnet and the default gateway for these devices should be the internal IP address of the local firewall or router, 192.168.0.1.

 

All connections initiated from the Internet can be blocked by the firewall except HTTPS traffic (TCP port 443). HTTPS traffic should be forwarded to the EasyAccess appliance address, 192.168.0.2. If TCP Tunneling will be enabled on the SSL VPN gateway, additional types of traffic initiated on the Internet should be permitted and forwarded to the EasyAccess appliance, for example encrypted mail traffic.

 

Network Interface Configuration

 

From the Interface window, an administrator may enable and disable network interfaces and configure IP settings.

 

The Interface window will display up to four detected Ethernet interface cards. The administrator may define which interface will terminate SSL VPN connections. This is the interface port that will be used to access the web management interface and the SSL VPN portal.

 

To change the Ethernet interface that terminates SSL VPN connections:

 

  1. Select the SSL VPN Termination radio button for the desired interface.

  2. Click Submit to update the configuration. Then restart the EasyAccess software for the change to take effect.

 

IP Address Configuration

 

Configure interface IP settings by clicking the Edit link for the desired network interface in the Interface window.
