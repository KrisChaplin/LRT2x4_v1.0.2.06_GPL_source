Services Overview

The Services window is the default window displayed when you log into the SSL VPN portal. You may also access this window by clicking the Services option in the left navigation menu.

 

The SSL VPN applications and your application bookmarks will be displayed in the Services window.

 

EasyAccess services include:

 

  • Web Proxy

  • HTTPS Proxy

  • Telnet Client

  • SSH Client

  • FTP Client

 

For more information about an EasyAccess service, click one of the service links listed above.
