Domains

To configure EasyAccess authentication domains, click the Access option in the left navigation menu, then click the Domains option.

 

In order to create access policies, you must first create authentication domains. By default, the EasyAccess authentication domain is already defined. The EasyAccess domain is the EasyAccess internal user database.

 

Additional domains may be created that require authentication to remote authentication servers. EasyAccess supports Radius, LDAP, NT Domain, Kerberos and Active Directory authentication in addition to internal user database authentication.
