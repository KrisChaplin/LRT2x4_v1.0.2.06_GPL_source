Logging into the Portal

After the EasyAccess software has been installed and the IP settings configured, you may log into the EasyAccess web user interface from a web browser.

 

The web user interface can be accessed locally from the EasyAccess gateway or remotely from a device connected to the LAN or WAN port of the EasyAccess gateway. The machine used for management is referred to as the “Management Station”.

 

  1. To log into the SSL VPN portal, enter the following into the Address or Location field of your web browser:

    https://[IP ADDRESS/DOMAIN NAME OF EasyAccess SERVER]
     

  2. An Authentication window with a Password dialogue box will be displayed.
     

  3. Enter the default administrator User Name and Password and select the EasyAccess Domain to initially log into the web user interface:

    User Name:     admin
    Password:     admin123
    Domain:     EasyAccess

    If you have already changed the default administrative password, enter your new password in the Password field.

    NOTE:     Both the user name and password are case-sensitive.
     

  4. Click Login to log into the EasyAccess web user interface.
     

  5. Until an SSL certificate is uploaded to the EasyAccess web server, the web browser may display a warning message. This message can be ignored during initial login. Please refer to the EasyAccess Installation Guide for SSL certificate management instructions.

 

If you cannot log into the EasyAccess system, click the Reload or Refresh button on the web browser and try again.

 

Once you have logged into the SSL VPN portal, an authenticated user session will be established. If you close the browser window, you will need to re-authenticate in order to log into the SSL VPN portal again.
