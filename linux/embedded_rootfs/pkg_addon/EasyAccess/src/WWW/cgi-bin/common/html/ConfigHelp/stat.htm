Status

To view the EasyAccess Status window, click the General option in the left navigation menu, then click the Status option.

 

From the Status page, you may view:

 

  • The EasyAccess software version

  • The processor (CPU) of the SSL VPN gateway

  • The amount of RAM memory on the SSL VPN gateway in MegaBytes (MB)

  • The available disk space of the SSL VPN gateway in GigaBytes (GB)

  • The uptime, the length of time since the SSL VPN gateway has been rebooted

  • The start time, the time and date since the EasyAccess SSL VPN software was last started

  • The number of active users. The number of active users includes administrative users. Click View current users or go to the Current Users page to view the list of current users.

  • The WAN IP address of the SSL VPN gateway. This is the equivalent of the eth0 interface of the SSL VPN gateway.

  • System Serial ID based on the hardware of the SSL VPN gateway. This ID is required to create an upgrade license key.

  • License information, such as the number of valid users licensed.
     

The Status page also displays a System Status dashboard with Memory and CPU utilization. Use the dashboard dials to view and learn how much of your system’s capacity you are using. This information will tell you whether you may need to upgrade your hardware as your number of concurrent users grows. The Memory and CPU dials show the percentage of available memory and processing power currently being used.

 

NOTE:  Both the Memory and CPU dashboard dials use Macromedia Flash.

 

In addition, the Status page includes two graphs that show bandwidth usage and concurrent users. The Bandwidth Usage graph displays the total inbound and outbound bandwidth in Kbps over the last six hours. The outbound data indicates files or data sent to SSL VPN users from the SSL VPN gateway or from the local network that the SSL VPN gateway resides. The inbound data indicates files uploaded by SSL VPN users or requests or data sent by SSL VPN users to the SSL VPN gateway.

 

The Active Users graph displays the total number of active users at any point in time over a six hour period. The number of active users is recorded every four minutes. Note that administrators are also included in the Active Users graph.
