Active Users

The Active Users window displays the active users and administrators logged into the SSL VPN portal or the EasyAccess administrative interface. To view the Active Users log file, click Status and Logs in the left navigation menu and then click Active Users.

 

Each entry displays the name of the user, the group in which the user belongs, the IP address of the user and a time stamp indicating when the user logged in.

 

An administrator may terminate a user session and log the user out by clicking the Delete (X) icon at the right of the user row.

 

To view the Active connections for each user:

 

To view the active connections for each user, click the Username in the Status and Logs >> Active Users window. The page displays the  type of Application, IP Address and its corresponding Port Number.

 

An administrator may terminate an active connection by clicking the Delete (X) icon at the right of the active connection row.

 

Note:Active connection will be displayed for Virtual Transport only after a service is launched.  For eg., after launching Virtual Transport if Telnet connection is established then the Application will be Virtual Transport and the Port Number is displayed as '23' in the Active connections page.
