Date

Configure the EasyAccess time and date settings by clicking General in the left navigation menu and then clicking Date.

 

EasyAccess uses the time and date settings to timestamp log events and for other internal purposes.
 

  1. Select your time zone in the Select Your Time Zone drop down menu.
     

  2. Check Automatically adjust for Daylight Saving Time checkbox, to enable DST.

     

  3. To manually define the time and date settings, enter the desired time (in 24-hour time format) and the date.

     

  4. Click the first Submit button to update the configuration.
     

NTP Configuration

 

To set the time of the EasyAccess device using the Network Time Protocol (NTP), perform the following steps. If you enable NTP, then the NTP time settings will override the manually configured time settings. The NTP time settings will be determined by the NTP server and the time zone that is selected in the Select Your Time Zone menu.

 

  1. Check the Automatically synchronize with an NTP server checkbox.
     

  2. Enter the time interval in seconds to synchronize time settings with the NTP server in the Update Interval field. If no period is defined, EasyAccess will select the default update interval, 64 seconds.
     

  3. Enter the NTP server IP address or fully qualified domain name in the NTP Server Address field.
     

  4. For redundancy, enter a backup NTP server address in the NTP Server Address (Optional) fields.
     

  5. Click the second Submit button to update the configuration.
