FTP Client

To access an Intranet FTP server through the SSL VPN portal, click on the FTP icon in the My Services table. A new browser window will be displayed.

 

     1.  Enter the name or IP address of the FTP server in the Address field.

 

     2.  Click Submit.

 

If the SSL VPN gateway is able to successfully connect to the FTP server, you will see a link such as "user@10.0.0.10" in the left navigation menu of the FTP client window. You can directly begin uploading and downloading files by clicking the Upload Files or Download Files options in the navigation menu.

 

If you do not see an active session, then you will need to re-authenticate. In the FTP New Session window, enter your FTP Server Name, User Name and Password and click Submit. You can also use the New Session window to log into several FTP servers at the same time.

 

Once you have logged into the FTP server, a web-based FTP client will allow you to download and upload files, make directories, rename files and delete files on the remote server.

 

If you see an error page that displays “Error: You are not authorized to access this server using this SSL-VPN service” then you do not have permission to access this host using FTP. Contact your EasyAccess administrator for more information.

 

If the window displays an error message, then the FTP server may be unavailable.
