Virtual Passage Installation for MacOS X

Installation Procedure

 

To install Virtual Passage and connect to the SSL VPN gateway, perform the following instructions:

 

    1. Click the Install Virtual Passage icon to download the Virtual Passage software file to your system.
       

    2. Once the file has been downloaded, double click on it to install the program. Install the software to the default location.
       

    3. Click the Create a Virtual Passage icon to open a connection to the SSL VPN gateway. A Dialog box will be displayed.
       

    4. Click Connect to open a Virtual Passage connection. Virtual Passage will transparently connect to the SSL VPN gateway and the remote network.

 

To end the SSL VPN Virtual Passage connection, click Disconnect. The Virtual Passage connection will take approximately 10 seconds to terminate.

 

NOTE: SSL VPN users can only create a Virtual Passage to the remote network during an active SSL VPN session. Once a user logs out of the SSL VPN portal or closes the portal window, the Virtual Passage session will terminate.
