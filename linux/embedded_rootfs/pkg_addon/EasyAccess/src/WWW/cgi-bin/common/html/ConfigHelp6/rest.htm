Restart

Restart the EasyAccess software daemons by clicking General in the left navigation menu and then clicking Restart.

 

  1. To reboot the FireBase device, click Reboot.
     

  2. You will be prompted to confirm the reboot. Click Yes to confirm the reboot.
     

The EasyAccess server may take up to 2 minutes to reboot, depending upon the hardware configuration of the system. While the server is being restarted, all sessions to the EasyAccess server will be dropped. SSL VPN users will need to authenticate again in order to access the EasyAccess SSL VPN portal.
 

Most EasyAccess configuration changes only require a restart to the EasyAccess software.
 

  1. To restart the EasyAccess software, click Restart.
     

  2. You will be prompted to confirm the restart. Click Yes to confirm.
     

The EasyAccess server may about 25 seconds to restart. While the server is being restarted, all sessions to the EasyAccess server will be dropped. SSL VPN users will need to authenticate again in order to access the EasyAccess SSL VPN portal.
