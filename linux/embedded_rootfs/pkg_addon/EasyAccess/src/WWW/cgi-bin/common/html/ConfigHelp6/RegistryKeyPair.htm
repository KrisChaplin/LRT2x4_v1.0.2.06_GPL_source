RegistryKeyPair

To view the EasyAccess Registry Key Pair window, click the EPS option in the left navigation menu, then click the Registry Key Pair option.

 

Add a Registry Key Pair

 

To create a new Registry Key, click Add Registry Key Pair. An Add Registry Key Pair window will be displayed.

 

          1. Enter the Name, Registry Key Name, Registry Key value name and Registry Key value that should be present

              in the EPS Client to logging in to EasyAccess.

          2. Click Submit.

 

 Example:

RegistryKey Name

HKEY_CURRENT_USER\Software\Filezilla

 

RegistryKey Value Name

Install_Dir

 

RegistryKey Value

C:\Program Files\FileZilla

 

Add a Registry Key Pair List

 

To create a new Registry Key Pair List, click Add Registry Key Pair. An Add Registry Key Pair List window will be displayed.

 

          1. Enter the Registry Key List Name in the Name field which contains list of Registries Keys to be defined for EPS

              policy.

          2. Select the various Registry Keys from the multiple select combo box.

          3. Click Submit.
