Company Logo

Import and display new logos in the EasyAccess administrative interface by clicking General in the left navigation menu and then clicking Company Logo.

 

EasyAccess supports multiple ways to customize the web user interface. These methods include importing new logos, defining a web browser title message, displaying a banner message and customizing the pages and applications that are visible to end users. Most of the customization features are defined in the Access >> Portal Layout section of the interface. Additional configuration, such as policies and bookmarks, are defined in Access >> Users and Groups.
 

To change the EasyAccess logo:

 

  1. Click the Browse button for Logo 1 and locate a GIF image file on your hard drive. The image should be approximately 173 pixels in width and 32 pixels in height.
     

  2. Click Browse for Logo2 and browse to find a GIF image that is approximately 170 pixels by 22 pixels.
     

  3. Click Browse for Logo3 and browse to find a GIF image that is approximately 128 pixels by 38 pixels.
     

  4. Click the Upload button to upload the new logos.
     

  5. Select Company Logo from the Use menu and click Submit to update the configuration. You may need to refresh your web browser window to view the new logo.

 

You can also upload each image separately if necessary. The first image, Logo1, is displayed in the default layout in the top, left corner. The second image is shown in the login page. The third image is displayed in the top menu layout—it is not visible if you choose to display the default layout with a left menu bar.

 

It is recommended that you upload logos with transparent backgrounds. In addition, the logo should be a dark color, because it will be shown against light backgrounds.
