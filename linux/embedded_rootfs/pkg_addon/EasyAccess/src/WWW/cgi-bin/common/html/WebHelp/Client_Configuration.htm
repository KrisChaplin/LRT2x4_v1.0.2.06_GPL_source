Client Configuration

The following is an example configuration for a Microsoft Outlook XP client. Your network administrator should provide you with the correct SMTP and POP3 mail server address, TCP port numbers and email address information.

 

  1. From the Tools menu in Microsoft Outlook, select Email Accounts.
     

  2. Select Add a new e-mail account and then click Next.
     

  3. Select POP3 server type and then click Next.

    NOTE: Other types of mail servers other than POP3 are supported, although not all IMAP extensions may be properly relayed through the SSL VPN gateway.
     

  4. Enter your name in the Your Name field.
     

  5. Enter your email address in the E-mail Address field.
     

  6. Enter your user name and password in the User Name and Password fields.
     

  7. Type the name or IP address of your mail server in the POP3 and SMTP fields. This could be the address of your mail server or the public address of the SSL VPN gateway.
     

  8. Click More Settings … and then click the Advanced tab at the top.
     

  9. For both POP3 and SMTP, check the This server requires a secure connection (SSL) checkbox.
     

  10. Enter the POP3 TCP port number in the Incoming server (POP3) field. This should be the same port number configured in the SSL VPN gateway for POP3 TCP tunneling.
     

  11. Enter the SMTP TCP port number in the Outgoing server (SMTP) field. This should be the same port number configured in the SSL VPN gateway for SMTP TCP tunneling.
     

  12. Click OK.
     

  13. If you are connected to the Internet, click Test Account Settings … to confirm that the email account was set up correctly.
