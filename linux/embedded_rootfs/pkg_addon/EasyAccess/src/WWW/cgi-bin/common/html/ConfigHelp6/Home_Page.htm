Home Page

To configure the portal Home Page window, click Access and then Portal Layouts in the left navigation menu. Then click on a Portal Layout name to edit the portal layout. In the portal layout window, click the Home Page Edit link to edit the SSL VPN portal Home page.

 

The SSL VPN portal home page enables you to create a custom page that mobile users will see when they log into the portal. Because the home page is completely customizable, it provides the ideal way to communicate remote access instructions, support information, technical contact info or VPN-related news updates to remote users. The home page is also well-suited as a starting page for restricted users; if mobile users or business partners are only permitted to access a few files or web URLs, the home page allows you to only show those relevant links to these users.

You can edit the title of the page, create a home page message that displays at the top of the page, show all applicable bookmarks (user, group, and global) for each user, and optionally upload an HTML file.

 

Editing the Portal Home Page

 

To edit the home page for the portal, perform the following steps:

 

  1. Enter a title in the Home Page Title field. The title will be displayed at the top of the portal home page. The maximum length of the title is 128 characters.

  2. Check the Display home page message checkbox to display a custom message at the top of the portal home page.

  3. Check the Display Desktop/Services bookmark table checkbox to display all applicable user, group and global bookmarks in a single table on the home page.

  4. If the Display home page message checkbox is checked, then enter a custom message to be displayed at the top of the portal home page in the Home page message field. Enter a plain text message or include HTML and JavaScript tags. The maximum length of the home page message is 4096 characters.

  5. Enter a title for the Desktop/Services bookmark table in the Title of bookmark table field. The maximum length of the title is 128 characters.

  6. In the Upload HTML file (optional) field, click Browse to locate and select an HTML file to upload. The maximum size of the uploaded HTML file is 25 KB.

  7. Click Submit to update the home page content.

Important Information about the portal Home Page

 

For most SSL VPN administrators, a plain text home page message and a list of links to network resources will provide the prefect portal home page. But for the more advanced administrator that wants to display additional content, please review the following information:

 

  • The home page is displayed in an IFRAME--internal HTML frame

  • The width of the iframe is 542 pixels, but since there is a 29 pixel buffer between the navigation menu and the content, the available workspace is 513 pixels.

  • You can upload a custom HTML file which will be displayed below all other content on the home page. You can also add HTML tags and JavaScript to the Home Page Message.

  • Since the uploaded HTML file will be displayed after other content, do not include <head> or <body> tags in the file.

 

Default Portal Home Page Message

 

The default SSL VPN portal home page includes a home page message. It demonstrates nested tables, images, and the standard table layout used in the SSL VPN portal. However, it is also possible to delete the default home page message and write a standard text message with no HTML tags.

 

More information and sample portal home page messages and HTML files are available at the Cavium Networks Support Site.
