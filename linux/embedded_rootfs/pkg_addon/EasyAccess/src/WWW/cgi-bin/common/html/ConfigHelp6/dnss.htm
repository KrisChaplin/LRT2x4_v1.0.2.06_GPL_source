DNS Settings

The DNS Settings window enables the EasyAccess administrator to configure the hostname, DNS server addresses and WINS server addresses. The WINS server configuration is optional; the DNS server configuration is required.

 

To configure the EasyAccess hostname, DNS settings and WINS settings:
 

  1. Enter the hostname for the EasyAccess device in the EasyAccess Hostname field. The hostname is used to identify the EasyAccess device on the network. Use only letters and numbers for the hostname; do not enter non-ASCII characters such as spaces or apostrophes.
     

  2. Enter the IP address of your DNS server in the Primary DNS Server field. It can also be an IPv6 address.
     

  3. For redundancy, enter the IP address of a backup DNS server in the Secondary DNS Server field. It can also be an IPv6 address.
     

  4. Enter the domain name of your network in the DNS Domain field. This field is optional and is not required for most network environments.
     

  5. Enter the IP address of your WINS server in the Primary WINS Server field. IPv6 address is also supported.
     

  6. Enter the IP address of a backup WINS server in the Secondary WINS Server field. IPv6 address is also supported.
     

  7. Click Submit to update the configuration. Once the default route has been added, the route will be displayed in the Current Routes table.

 

NOTE: If you update the EasyAccess hostname, you will need to restart the EasyAccess software for the change to take effect. DNS and WINS settings changes take effect immediately.
