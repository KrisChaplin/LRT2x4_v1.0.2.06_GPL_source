Logging into the Portal

Users may log into the EasyAccess SSL VPN portal from any computer with Internet connectivity. No additional software other than a standard web browser with Java enabled is required.

 

The SSL VPN portal provides remote network access using HTTP and Java applets and all network traffic between the client and the EasyAccess gateway are transmitted through a secure SSL tunnel.

 

  1. To log into the SSL VPN portal, enter the following into the Address or Location field of your web browser:

    https://[IP ADDRESS/DOMAIN NAME OF EasyAccess SERVER]
     

  2. An Authentication window with a Password dialogue box will be displayed.
     

  3. Enter the user name provided to you by your network administrator in the User Name field.
     

  4. Enter your password provided to you in the Password field. Both the user name and password are case-sensitive.
     

  5. Select your domain in the Domain menu.
     

  6. Click Login to log into the EasyAccess SSL VPN portal.
     

If an SSL certificate warning message is displayed, contact your EasyAccess administrator. Your EasyAccess administrator may need to install an authorized SSL certificate on the SSL VPN gateway or configure your web browser or Java settings to accept the current SSL certificate.

 

If you cannot log into the EasyAccess system, click the Reload or Refresh button on the web browser and try again.

 

Once you have logged into the SSL VPN portal, an authenticated user session will be established. If you close the browser window, you will need to re-authenticate in order to log into the SSL VPN portal again.

 

NOTE: If you open a new browser window to access an application from within the SSL VPN portal and then you close the SSL VPN portal window, then the new application window will lose connectivity to the SSL VPN gateway. You will need to log into the SSL VPN portal again in order reestablish a secure SSL VPN session.

 

The SSL VPN portal consists of Home, Services, Applications, Desktop, and Network Places, Virtual Passage and Themes sections where you can access a desired file, application or other resource.

 

NOTE: The administrator can customize the SSL VPN portal. So all applications listed in online help may not be available.

 

Through the SSL VPN portal, you can connect to:

 

  • Intranet web or HTTPS sites – If your organization supports web-based email, such as Outlook Web Access, you can also access web-based email

  • FTP servers

  • Your network neighborhood for file sharing

  • Telnet and SSH access to servers and desktops

  • Your desktop or server

  • Specific applications installed on your desktop or on an application server

  • All network resources using the Virtual Passage SSL VPN client

 

The Themes page allows you to customize the color scheme of the portal.

 

The EasyAccess administrator determines what resources are available to users from the SSL VPN Portal. The EasyAccess administrator can create user, group, and global policies that disable access to certain machines or applications on the corporate network.

 

The EasyAccess administrator may also define bookmarks, or preconfigured links, to web sites or computers on the Intranet. Additional bookmarks may be defined by the end user.
