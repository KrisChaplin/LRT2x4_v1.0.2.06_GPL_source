VNC (Virtual Network Computing) Access

To connect to a desktop or server with VNC software installed, click on the VNC icon in the Remote Desktop Access table. A new browser window similar to the following will be displayed.

 

    1. Enter the domain name, host name or IP address of the VNC server in the Address field.
       

    2. Click Submit.

 

NOTE: VNC also supports accessing remote machine through user desired ports by entering the port number along with the IP address of VNC server. eg. 192.168.1.1:3001.

 

If the SSL VPN gateway is able to successfully connect to the specified server, the remote desktop will be displayed.

 

Do not close the window that opened the VNC desktop window; closing this window will close the session to the remote desktop.

 

Once you have logged in, you will be able to open files and applications, make file changes and save them on the remote machine.

 

If you see an error page that displays “Error: You are not authorized to access this server using this SSL-VPN service” then you do not have permission to access this host using VNC. Contact your EasyAccess administrator for more information.

 

If the window displays an error message, then the server may be unavailable or it may not be correctly configured to accept VNC connections.
