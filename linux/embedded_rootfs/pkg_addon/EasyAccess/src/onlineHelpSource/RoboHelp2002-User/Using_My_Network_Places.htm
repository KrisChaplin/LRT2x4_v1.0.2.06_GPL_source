Using My Network Places

You can access the My Network Places window by logging into the SSL VPN portal and then clicking the My Network Places option in the left navigation menu. A new Network Neighborhood window will be opened.

 

Your network places, or Windows file sharing workgroups, will be displayed. Click on the name of a workgroup to view the machines in that group. Then click on the name of the machine that you wish to view. Continue until you have accessed the folders and files that you desire.

 

EasyAccess will automatically browse network shares as an anonymous user and also supply your SSL VPN portal user name and password to authorize access to file sharing resources. If your EasyAccess user name and password do not grant you access to desired servers or shares, a login window will be displayed.  Enter the server login name and password and click Login.

 

NOTE: If your web browser blocks popup windows, then the file sharing login window may be blocked. Enable popup windows in order to manually log into fileservers and shares.

 

From the My Network Places window, you can navigate through the entire network, upload and download files and make new folders on a remote machine.

 

To navigate and browse network shares, click the name of a workgroup, server or share from the list of network places. Or enter the name or IP address of a server preceded by two backslashes (\\) in the Network Places Address Bar.

 

Once you have logged into a fileserver and opened a directory share, a window similar to the following will be displayed.

 

Download Files

 

Download files by clicking the name of the file from the list of files. Then choose to open the file directly or download the file to disk.

 

Upload Files

 

Upload files by clicking the Browse button in the left side of the browser window, selecting a local file from your hard drive and clicking Submit.

 

Delete Files

 

Delete files by checking the checkbox(es) next to the name of the file(s) you wish to delete. Then click Delete at the top of the window. A confirmation dialogue window will be displayed. Click Yes to delete the selected files.

 

Rename Files

 

Rename files by checking the checkbox(es) next to the name of the file(s) you wish to rename. Then click Rename at the top of the window. A new Rename Files window will be displayed.

 

In the Rename Files window, enter the new name of the file in the New Name field and then click Submit.

 

Click Logout to end the file server connection and close the Network Places window.
