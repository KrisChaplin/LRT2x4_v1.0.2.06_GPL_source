Making the SSL VPN Portal a Trusted Site

If your Internet Explorer security settings are set to "High" or prevent the EasyAccess ActiveX or Java applications from running properly, you may need to add the SSL VPN portal to your browser's list of Trusted Sites.

 

To ensure that your machine is protected against malicious code and spyware files, it is not recommended to change your web browser’s security settings in order to use the Java and ActiveX files included in the EasyAccess SSL VPN portal. Instead, add the SSL VPN portal address to your web browser’s list of Trusted sites.

 

Most web browsers today support multiple security zones with different permission levels. Trusted sites have lower security settings that will, by default, allow the EasyAccess Java and ActiveX content to function properly.

 

NOTE: ActiveX content is only supported in Internet Explorer.

 

Adding a Trusted Site in Internet Explorer

 

To add the EasyAccess SSL VPN portal as a trusted site:
 

  1. Go to the SSL VPN portal login page.
     

  2. Press ALT+D to select the SSL VPN portal address, and press CTRL+C to copy it to the Windows Clipboard.
     

  3. On the Internet Explorer Tools menu, click Internet Options.
     

  4. Click the Security tab, then click Trusted Sites. The Internet Options Security configuration window will be displayed.
     

  5. Click the Sites button. The Trusted Sites dialog window will be displayed.
     

  6. In the Add this Web site to the zone box, press CTRL+V to paste the SSL VPN Portal web address.
     

  7. Click the Add button.
     

  8. Click OK twice.
