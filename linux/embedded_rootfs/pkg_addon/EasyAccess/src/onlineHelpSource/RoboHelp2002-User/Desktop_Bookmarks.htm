Desktop Bookmarks

EasyAccess bookmarks provide a convenient way for you to access machines on the remote network that you will connect to frequently. To define desktop bookmarks, click Add Bookmark. An Add Bookmark window will be displayed.

 

 1.  To create a bookmark, enter a descriptive name in the Bookmark Name field.

 

 2.  Enter the domain name or the IP address of a host machine on the LAN in the Name or IP Address field.

 

 3.  Select the service type in the Service menu.

 

 4.  Enter the path of the application on the remote server (for example, C:\Program Files\Microsoft Office\Office10\EXCEL.EXE) in the Application field.

 

NOTE: The application path only needs to be defined for individual services. Leave this field blank to access the entire desktop.

 

 5.  Click Submit to update the configuration.

 

Once the configuration has been updated, the new bookmark will be displayed in the Bookmarks table. Click on a bookmark description to go to the bookmark location that you defined.

 

Delete a Bookmark

To delete a bookmark, click the X (delete) icon in the Bookmarks table for the bookmark you wish to remove. Once EasyAccess has been updated, the deleted bookmark will no longer be displayed in the Bookmarks table.
