SSL VPN Portal Theme

To view the Settings window, click the Settings option in the left navigation menu.

Select a Portal Theme

Users can select individual themes for the EasyAccess portal. To select a theme:

 

  1. Select the desired theme in the Select Theme menu. A preview of the theme will be displayed.
     

  2. Click the Submit button to change the portal theme.
     

The web page will be refreshed and the new theme will be displayed.

 

NOTE: Selecting a theme will only affect the layout and color scheme. The selection will not affect the portal's content or the layout for other users. Themes are not applicable for External domain users.
