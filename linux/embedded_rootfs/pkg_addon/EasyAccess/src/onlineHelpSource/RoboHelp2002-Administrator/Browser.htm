Browser

To view the EasyAccess Browser window, click the EPS option in the left navigation menu, then click the Browser option.

 

Add a Browser

 

To create a new Browser, click Add Browser. An Add Browser window will be displayed.

 

         1. Select the Browser name and Service Pack from the Browser dropdown list that should be present in EPS Client to

             logging in to EasyAccess.

         2. Click Submit.

 

         Note: The following Browser and  its versions were supported in EasyAccess

                   Internet Explorer versions 6.0 and 7.0

 

Add a Browser List

 

To create a new Browser List, click Add Browser List. An Add Browser List window will be displayed.

 

         1. Enter the Browser List name in the Name field which contains list of browsers to be defined for EPS Policy.

         2. Select the various Browser Names from the multiple select combo box.

         3. Click Submit.
