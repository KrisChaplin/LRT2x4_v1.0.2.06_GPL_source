Host Resolution

The Host Resolution window enables network administrators to configure or map host names or fully qualified domain names (FQDNs) to IP addresses. Note that EasyAccess can act as both a NetBIOS and WINS (Windows Internet Naming Service) client to learn local network host names and corresponding MAC and IP addresses.

 

To configure host resolution:

 

  1. Enter the IP address of the machine that will be mapped to a host name in the IP Address field.
     

  2. Enter the host name or fully qualified domain name of the machine in the Host Name field. For example, enter “mycomputer” or “mailserver.mycompany.com”. Do not enter names with spaces or other non-ASCII characters such as apostrophes or commas.
     

  3. Enter the host alias in the optional Alias field. For example, if a full FQDN “mailserver.mycompany.com” has been entered in the Host Name field, then a shorter name, such as “mailserver” may be entered in the Alias field.
     

  4. Click Submit to update the configuration. Once the domain has been added, the domain will be added to the Configured Host Names table.
