Antivirus

  

To view the EasyAccess Antivirus window, click the EPS option in the left navigation menu, then click the Antivirus option.

 

Add an Antivirus

 

To create a new Antivirus, click Add Antivirus. An Add Antivirus window will be displayed.

 

          1. Enter the Antivirus name, version and Virus definition version that should be present in the EPS Client to

              logging in to EasyAccess.

          2. Select the Check box for Enable version check, Enable Scanning, Enable product up to date check and

              Enable Virus definition check.

          3. Click Submit.

 

          Note: The following Antivirus and its versions were supported in EasyAccess.

 

                   McAfee VirusScan   10.0, 11.2

                   Symantec AntiVirus   9.3, 10.1.4.4000

                   TrendMicro   14.10.1023, 15.3, 15.30.115, 15.10.1206,

                   Kaspersky   6.0.2.621, 6.0.2.678

 

 

Add an Antivirus List

 

To create a new Antivirus List, click Add Antivirus List. An Add Antivirus List window will be displayed.

 

          1. Enter the Antivirus List Name in the Name which contains list of Antivirus to be defined for EPS policy.

          2. Select the various Antivirus Names from the multiple select combo box.

          3. Click Submit.
