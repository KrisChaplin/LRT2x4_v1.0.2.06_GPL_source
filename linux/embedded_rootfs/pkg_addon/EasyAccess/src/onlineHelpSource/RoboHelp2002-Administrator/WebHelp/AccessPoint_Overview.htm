Logging Into the Portal

The SSL-VPN portal is a web based portal that enables you to access Intranet or Extranet resources from any location without installing and configuring client software. The SSL-VPN portal provides encrypted connectivity through a standard web browser, allowing mobile workers, partners and customers to authenticate and access files, applications, and internal web sites securely.

 

All communications between the web browser and the EasyAccess SSL-VPN gateway are performed by HTTP and Java. The network traffic between the client and the SSL-VPN gateway is secured through an encrypted SSL connection.

 

The SSL-VPN Portal enables you to browse Intranet web sites, access your network neighborhood, upload and download files, access remote desktops and applications, and use Telnet or SSH to manage remote devices.

 

Browser Requirements

 

To configure the EasyAccess SSL-VPN software, an administrator must use a web browser with Javascript, cookies, and SSL enabled.

 

For Windows operating systems

 

•  Browser: Internet Explorer 5.0 or higher

•  Java: Sun JRE 1.3.x or 1.4.x

 

For Unix operating systems

 

•  Browser: Mozilla 1.x

•  Java: Sun JRE 1.3.x or 1.4.x
