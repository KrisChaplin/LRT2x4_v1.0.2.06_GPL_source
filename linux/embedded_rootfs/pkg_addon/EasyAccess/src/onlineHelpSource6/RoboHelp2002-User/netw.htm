Network Places Overview

The My Network Places option enables users to browse the network neighborhood, open and download files, upload files, and rename files and directories. The My Network Places option may be accessed by clicking the My Network Places option in the left navigation menu.

 

My Network Places relies on the CIFS (Common Internet File System), a protocol for sharing data and files across Intranets and the Internet. CIFS is an enhanced version of the Microsoft open, cross-platform Server Message Block (SMB) protocol.
