Service Bookmarks

EasyAccess bookmarks provide a convenient way for you to access web, FTP, or other servers on the remote network that you will connect to frequently. To define application bookmarks, click Add Bookmark at the bottom of the Services window. An Add Bookmark window will be displayed.

 

  1. To create a bookmark, enter a descriptive name in the Bookmark Name field.
     

  2. Enter the domain name or the IP address of a host machine on the LAN in the Name or IP Address field.
     

  3. Select the application type in the Service menu.
     

  4. Click Submit to update the configuration.

 

Once the configuration has been updated, the new bookmark will be displayed in the Bookmarks table. Click on a bookmark description to go to the bookmark location that you have defined.

 

Delete a Bookmark

 

To delete a bookmark, click the X (delete) icon in the Bookmarks table for the bookmark you wish to remove. Once EasyAccess has been updated, the deleted bookmark will no longer be displayed in the Bookmarks table.

 

If a bookmark does not have the X (delete) icon, then it has been configured by your network administrator and it cannot be deleted.
