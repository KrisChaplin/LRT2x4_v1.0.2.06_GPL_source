EasyAccess Overview

EasyAccess SSL-VPN provides secure remote access to network resources, such as applications, files, Intranet web sites, and email, through a web browser or through client software, such as Microsoft Outlook. The underlying protocol used for these sessions is SSL.

 

With the SSL-VPN, mobile workers, telecommuters, partners and customers can access information and applications on your Intranet or Extranet. What information should be accessible to the user is determined by access policies configured by the EasyAccess administrator.

 

Remote network resources can be accessed in the following ways:

 

  • Using a standard web browser - To access network resources, you must log into the EasyAccess SSL-VPN portal. When authorized, you can access your desktop on the remote network, launch network applications, download and upload files, browse your network neighborhood for file sharing, use telnet or SSH for remote management, and access Intranet web and HTTPS sites.

 

  • Using client software - To connect to a remote server, you just need to launch client software, such as Microsoft Outlook, Outlook Express, IBM Lotus Notes or Netscape Mail. Your client software will automatically try to contact the remote server, transparently authenticate using the settings that you have defined in the client. The EasyAccess SSL-VPN gateway device will receive the request made by the client. Depending on the configuration, the EasyAccess SSL-VPN gateway will either forward the request in clear text or via SSL to the internal network server. Then you can download and send emails, download Lotus Notes calendar information, or access other application servers securely.
