Change Password

The Change Password window allows the user to change his login password.

 

  1. Enter the Existing Password in the Old Password field.

     

  2. Enter the new password in the New Password field.

     

  3. Re-enter the new password in the Confirm New Password field.

     

  4. Click Apply. Once the password is successfully updated, the user will be logged out and the user should re-login with his new password.
