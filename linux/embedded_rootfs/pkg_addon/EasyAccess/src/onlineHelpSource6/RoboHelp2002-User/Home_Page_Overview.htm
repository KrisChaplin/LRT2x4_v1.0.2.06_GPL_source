Home Page Overview

The Home page window is the default window displayed when you log into the SSL VPN portal. You may also access this window by clicking the Home option in the left navigation menu.

 

NOTE: The portal pages and content will vary based on the configuration of your network administrator. So some pages described in the EasyAccess SSL VPN User Guide may not be displayed when you log into the SSL VPN portal.

 

The SSL VPN portal Home page provides customized content and links to network resources. The Home Page may contain support contact information, VPN instructions, company news, or technical updates.

 

The Home page may also contain relevant links to protected network resources or public Internet content.

 

If a table of network resources is displayed, click the links to access Intranet web sites, Cifs bookmarks, FTP sites, Terminal Server desktops, Terminal Services applications, VNC, Telnet or SSH servers.

 

Only a web browser is required to access Intranet web sites and FTP sites. VNC, Telnet and SSH require Java. And Terminal Services requires either Java or ActiveX on the client machine.
