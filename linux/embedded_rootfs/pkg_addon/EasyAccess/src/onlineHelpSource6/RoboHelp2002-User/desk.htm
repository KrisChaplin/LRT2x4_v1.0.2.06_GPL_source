Desktop Overview

The My Desktop window enables you to access a remote desktop, desktop application, or a home directory on a central server. You may also access this window by clicking the My Desktop option in the left navigation menu.

 

Terminal Services (RDP) and Virtual Network Computing (VNC) icons will be displayed in the Desktop and Desktop Applications table. Your bookmarks will be displayed in the Bookmarks table.

 

Terminal Services

Terminal Services or RDP (Remote Desktop Protocol) is a protocol supported on Windows XP Professional and Windows 2000 and 2003 Server. Once Terminal Services has been configured, users can connect to the machine remotely, access files and software, save file changes and perform other advanced features. Terminal Services does not require software to be installed on the machine in the internal network. Terminal Services also allows multiple users to access a single machine simultaneously.

 

VNC

Virtual Network Computing (VNC) is similar to Terminal Services, but it does not support all of the advanced features available with Terminal Services. Only one machine may log in and control a remote desktop at a time. And VNC software must be installed on the machine in the internal network in order to support remote desktop access. But with VNC, users can access machines running most operating systems including Windows, Mac OS, Linux, and Solaris.

 

Both Terminal Services and VNC support the unique ability to launch individual applications running on a remote desktop or server. Just running a single application reduces the latency and overhead compared to accessing the entire remote desktop. VNC server software may be downloaded for free at sites such http://www.realvnc.com or http://www.tightvnc.com.
