SSH Client

To open an SSH session, click on the SSH icon in the My Services table. A new browser window will be displayed.

 

   1.  Enter the name or IP address of the SSH server in the Address field.

 

   2.  Click Submit.

 

If the SSL VPN gateway is able to successfully connect to the specified server, a small login window will be displayed and you will be prompted to enter a user name and password.

 

Troubleshooting Information

 

If the SSH login window does not appear, then the login window may be hidden behind the SSH terminal window. Move or minimize the SSH Terminal window.

 

If you enter data into the SSH window, but data does not appear to be accepted, click your mouse on the white cursor in the SSH window.

 

If you see an error page that displays “Error: You are not authorized to access this server using this SSL-VPN service” then you do not have permission to access this host using the EasyAccess Web proxy application. Contact your EasyAccess administrator for more information.

 

If the window displays an error message, then the SSH server may be unavailable.
