Global Bookmarks

To edit Global Access Policies, click Global Policies in the Access >> Users and Groups window. The Edit Global Settings window will be displayed.

Edit Global Bookmarks

To define global bookmarks, click Add Bookmark. An Add Bookmark window will be displayed.

 

When global bookmarks are defined, all members will see the defined bookmarks from the EasyAccess SSL VPN portal. Individual users will not be able to delete or modify global bookmarks.

 

  1. To create a bookmark, enter a descriptive name in the Bookmark Name field.
     

  2. Enter the domain name or the IP address of a host machine on the LAN in the Name or IP Address field.
     

  3. Select the service type in the Service menu.
     

  4. Click Submit to update the configuration.

 

Once the configuration has been updated, the new global bookmark will be displayed in the Global Bookmarks table.
