Web Browser

The EasyAccess SSL VPN portal may be accessed through a standard web browser. The following are the recommended web browsers for the EasyAccess SSL VPN portal.

 

Web browser requirements for Windows operating systems:

 

  • Internet Explorer 5.0 or higher*

  • Mozilla 1.0 or higher

  • Netscape 7.0 or higher

  • Opera 7.0 or higher

 

* Internet Explorer is required for ActiveX clients such as Virtual Passage and Virtual Transport.

 

Web browser requirements for MacOS operating systems:

 

  • Safari 1.0 or higher

 

Web browser requirements for Linux and Solaris operating systems:

 

  • Mozilla 1.0 or higher

  • Netscape 7.0 or higher

 

Web browser configuration:

 

  • JavaScript must be enabled

  • Cookies must be enabled

  • SSL (HTTPS) must be enabled

  • Java must be enabled
