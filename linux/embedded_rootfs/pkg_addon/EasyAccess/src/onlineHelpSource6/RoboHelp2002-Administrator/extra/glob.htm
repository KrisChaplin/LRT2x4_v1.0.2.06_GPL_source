EasyAccess Global Settings

To view the EasyAccess Themes window, click the General and then the Themes option in the left navigation menu.

Select an EasyAccess Global Theme

The administrator may select individual themes for the EasyAccess portal. The configured theme will be seen by all users who have not configured an individual theme in the SSL VPN portal Settings window.

To select a theme:


  1. Select the desired theme in the Select Theme menu. A preview of the theme will be displayed.
     

  2. Click the Submit button to change the portal theme.
     

The web page will be refreshed and the new global theme will be displayed.

In addition, the administrator may select the layout of the SSL VPN portal. The default layout is a centered, framed window. The administrator may optionally configure a top menu layout. The layout change will affect all users and is not customizable by SSL VPN users.
