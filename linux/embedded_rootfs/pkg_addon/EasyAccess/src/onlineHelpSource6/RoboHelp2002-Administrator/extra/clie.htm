Virtual Passage Client Addresses

To view the Virtual Passage Client Addresses window, click the Virtual Passage option in the left navigation menu.

Define Client Addresses

When remote users connect using the Virtual Passage SSL VPN client, they will receive dynamically assigned IP and DNS settings for their virtual network interfaces. You may define the IP address range to assign to incoming Virtual Passage clients on the Client Addresses window.
 

NOTE: It is recommended to assign an IP address range in the same subnet as the remote network. But be sure that there are no overlapping addresses between client addresses and devices on the remote network.


  1. Enter the first IP address of the IP address range in the Client Address Range Begin field.
     

  2. Enter the last IP address of the IP address range in the Client Address Range End field.
     

  3. Click Submit to update the configuration.
