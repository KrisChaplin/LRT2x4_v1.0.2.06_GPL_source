Layout

To view the EasyAccess Layout window, click the General and then the Layout option in the left navigation menu.

Upload a corporate logo

To upload your company logos, select Browse for Logo 1 and navigate to the location on your disk or shared drive where your logo is saved. Then select the GIF logo. Perform the same procedure for Logo 2 and Logo 3.

The size of the three logos are 173 x 32 pixels, 170 x 22 pixels, and 128 x 38 pixels in width and height respectively.

Click Upload to upload the three logos.

Then select Use Company Logo from the drop down menu and click Submit.

Edit Desktop Application Path

You can set the path on the Terminal Services server where Microsoft Word and Excel are installed. Then users will be able to launch these applications directly. To set the path:

  1. Enter the path on the Terminal Services server where Microsoft applications are installed in the Word/Excel Path field.
     

  2. Click the Submit button to change the path.
     

Note that applications must be installed especially for Terminal Services in order to support this feature. See the Menlo Logic technical note on this procedure for more information.

Customize SSL VPN Portal

You may also configure the title that is displayed at the top of the web browser window in the SSL VPN portal and add a custom message in the login page.

  1. Enter the title to be displayed at the top of the web browser window in the Web Page Title field.
     

  2. Select whether you wish to display a banner message on the login page in the Login Page menu.
     

  3. If you wish to display a banner message, then enter the title of the message in the Banner Title field.
     

  4. Enter the banner text to be shown on the login page in the Banner Text field.
     

  5. Click the Submit button to update the settings.
