IP Settings

EasyAccess requires static IP configuration. To configure the EasyAccess IP settings, complete the following instructions:

 

  1. Enter a unique, valid IP address from your network address range in the IP Address field.

    If this network interface is configured to terminate SSL VPN connections, then this will be the IP address of the EasyAccess web management interface and the SSL VPN portal.
     

  2. Enter your network subnet mask in the Subnet Mask field.

    The subnet mask identifies which IP addresses are directly accessible to the EasyAccess device and which addresses are reached through a router. An example of a valid subnet mask that supports up to 254 addresses is “255.255.255.0”.
     

  3. Select whether you wish to enable or disable the interface in the Interface Status menu.

    You may not disable the interface if the interface is configured to terminate SSL VPN connections.
     

  4. Click Submit to update the configuration.

    If the interface is configured to terminate SSL VPN connections, then restart the EasyAccess software for the change to take effect.

 

NOTE: The EasyAccess administrative session will end when the software is restarted. To login to the EasyAccess management interface, enter the new IP address of the EasyAccess device in the Address or Location field of your web browser. Be sure that the management station is in the same subnet as the new EasyAccess IP address.  

 

To complete the IP settings configuration, also configure EasyAccess DNS settings and network routes.
