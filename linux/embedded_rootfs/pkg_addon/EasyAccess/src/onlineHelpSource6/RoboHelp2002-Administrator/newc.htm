Generate New CSR

In order to get a valid certificate from a widely accepted Certificate Authority such as Verisign or Thawte, you must generate a Certificate Signing Request for your SSL VPN device.

 

  1. To generate a CSR, click General in the left navigation menu, then click Certificates.
     

  2. Click New CSR/CRT  to generate a new Certficate Signing Request (CSR) or a Self-signed Certificate (CRT). The Create CSR window will be displayed.
     

  3. Complete the New CSR/CRT  form and click Submit.
     

If all information is entered correctly, a csr.zip/crt.zip file will be created. The file will include a server.csr signing request  and a server.key key file for CSR and a server.crt and server.key file for CRT. Save the zip file to disk. You will need to provide these files to the Certificate Authority.
