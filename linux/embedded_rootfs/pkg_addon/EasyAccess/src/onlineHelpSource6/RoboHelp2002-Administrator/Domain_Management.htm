Domain Management

Current Domains Table

All of the configured Domains are listed in the Current Domains table in the Domains window. The Domains are listed in the order in which they were created.

Delete a Domain

To delete a domain, click the X (Delete) icon in the Current Domains table for the domain you wish to remove. Once EasyAccess has been updated, the deleted domain will no longer be displayed in the Current Domains table.

 

NOTE: The EasyAccess domain cannot be deleted.
