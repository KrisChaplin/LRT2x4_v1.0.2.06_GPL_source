Java Requirements

Several of the SSL VPN applications require Java thin client support, but it is not required to log in and use the SSL VPN portal.

 

Java requirements for Windows, MacOS or UNIX operating systems:

 

  • Sun JRE 1.1 or higher and MS JVM 5 or higher

 

Java support is required for Telnet, SSH, and VNC remote desktop applications. It is not required for web-based FTP, Web Proxy, HTTPS Proxy, and My Network Places/file sharing applications. It is also not required for Virtual Passage or Virtual Transport ActiveX clients.

 

Users can download the latest Sun Java plugin at: http://www.java.com.

 

For instructions on enabling the Sun Java plug-in through the control panel, see:

 

http://www.java.com/en/download/help/enable_panel.jsp.

 

To enable the Sun Java plug-in through a web browser, see:

 

http://www.java.com/en/download/help/enable_browser.jsp.
