PATCH

To view the EasyAccess Patch window, click the EPS option in the left navigation menu, then click the Patch option.

 

Add a Patch

 

To create a new Patch, click Add Patch. An Add Patch window will be displayed.

 

1. Enter the Patch name in the Patch field that should be present in the EPS Client to logging in to EasyAccess.

2. Click Submit.

 

    Example:

    Patch

    Security Update for Windows XP (KB 901214).

 

Add a Patch List

 

To create a new Patch List, click Add Patch List. An Add Patch List window will be displayed.

 

1. Enter the Patch List name in the Name field which contains the list of patches to be defined for EPS Policy.

2. Select the various Patches from the multiple select combo box.

3. Click Submit.
